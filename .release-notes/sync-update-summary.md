# 会话与 Skills 同步体验升级

## 新增功能

- ☁️ 会话同步现在会并排展示本地与云端状态，支持批量上传、删除云端副本，并在两端都发生变化时提示冲突。
- 📦 Skills 现在可以统一查看本地与云端版本，在详情中使用 Markdown 阅读内容、查看文件差异，并批量上传、下载或删除云端副本。
- 🔄 支持为 Claude Code 与 Codex 安装会话自动同步 Hook，每轮回复完成后只上传发生变化的会话，也可以随时关闭或移除。
- 🧭 首次配置或云端结构需要更新时，可以直接复制最新 SQL 并打开对应项目的 SQL Editor，执行后原地刷新继续使用。
- 🧹 新增 `agent-session-search uninstall`，可清理应用写入的集成配置和缓存，同时保留会话数据库、同步配置与用户偏好。
- 🇨🇳 首次打开应用时默认使用中文，已经手动选择过语言的用户会保留原设置。

## Bug 修复

- 修复 Windows 上启用 TClaude、TCodex 或 CodeBuddy CLI 会话来源后，远端刷新完成却无法拉取会话的问题。
- 修复重复上传同一会话仍被判断为内容变化，以及子智能体会话出现在同步清单中的问题。
- 修复同名 Skill 可能相互覆盖、大型 Skill 上传失败，以及上传完成后仍显示“正在上传”的问题。
- 修复关闭云端会话预览后直接退出同步清单，并改善会话恢复、上传、更新和删除操作的状态反馈。
- 修复应用更新失败后旧版本无法继续启动的问题，并在自动更新失败时提供手动安装入口。
