# 自定义供应商即刻用于 AI 摘要与搜索

## 新增功能

- Codex 自定义供应商的「模型」现在是一个统一的输入框：可以直接手动填写模型名，也可以点「探测模型」后从下拉建议里选择，不用再在两个控件之间来回确认。

## Bug 修复

- 配置并保存自定义 Codex 或 Claude 供应商后，AI 摘要与搜索会立即使用该自定义配置，不再需要单独打开摘要设置再保存一次。
- 取消或超时的 AI 摘要不会再意外触发第二次摘要请求，避免多余的等待和额度消耗。
- 部分新模型不再支持 temperature 参数时，AI 摘要会自动重试并成功返回，不再报「temperature is deprecated」错误而中断。
- 修正了 CodeBuddy 会话的 Token 统计：此前只统计了少量消息，导致显示的 Token 数远低于真实用量（例如几十万 vs 实际数千万）。现在会完整统计每次工具调用产生的用量，已有会话也会在升级后自动重新计算，显示与真实额度消耗一致的 Token 数。
