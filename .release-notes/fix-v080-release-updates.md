# v0.8.0：同步体验升级与更新恢复

## 新增功能

- ☁️ 会话同步现在会并排展示本地与云端状态，支持批量上传、删除云端副本，并在两端都发生变化时提示冲突。
- 📦 Skills 现在可以统一查看本地与云端版本，在详情中使用 Markdown 阅读内容、查看文件差异，并批量上传、下载或删除云端副本。
- 🔄 支持为 Claude Code 与 Codex 安装会话自动同步 Hook，每轮回复完成后只上传发生变化的会话，也可以随时关闭或移除。
- 🧭 首次配置或云端结构需要更新时，可以直接复制最新 SQL 并打开对应项目的 SQL Editor，执行后原地刷新继续使用。
- 🧹 新增 `agent-recall uninstall`，可清理应用写入的集成配置和缓存，同时保留会话数据库、同步配置与用户偏好。
- 🇨🇳 首次打开应用时默认使用中文，已经手动选择过语言的用户会保留原设置。

## Bug 修复

- 修复正式安装版被误显示为开发版本，导致应用不检查更新、没有新版本红点且无法从“关于”页面更新的问题。
- 修复终端更新时无法识别仍在运行的旧应用，可能因文件占用而安装失败的问题。
- 修复更新失败提示包含乱码或不可读控制字符的问题，并继续提供手动安装入口。
